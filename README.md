# AI Project Mentor

AI Project Mentor is an AI-powered platform that analyzes GitHub repositories and provides personalized feedback, improvement suggestions and project specific guidance to help developers build better software.

## Features

* Analyze GitHub repositories using a repository URL
* Review project architecture and code quality
* Detect security and performance issues
* Generate improvement suggestions
* Score projects based on quality metrics
* AI chat for project-specific questions
* Generate project-based interview questions
* Create personalized learning roadmaps

* Gemini API
* GitHub API
* JWT Authentication

##  How It Works

1. Submit a GitHub repository URL.
2. The system analyzes the repository structure and code.
3. AI generates:

   * Project review
   * Security analysis
   * Performance analysis
   * Improvement suggestions
4. Ask follow-up questions and receive project-specific guidance.

##  Goal

Help developers improve their projects, learn best practices and prepare for technical interviews through AI-powered repository analysis.
